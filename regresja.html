Analiza regresji

Jak opisać zależność liczbą i wzorem: model Y = a + b·X, parametry z metody najmniejszych kwadratów, ocena dopasowania (R², φ²) oraz prognozowanie z modelu i ocena trafności prognoz.

01

Materiały

Korelacja mówi, jak silnie zmienne idą razem. Regresja idzie krok dalej — opisuje kształt zależności równaniem i pozwala prognozować. Zmienną Y (zależną) modelujemy w oparciu o zmienną(e) X (niezależną).

Model regresji prostej

jedna zmienna objaśniająca
Y = a + b·X
  • Y — zmienna objaśniana (zależna)
  • X — zmienna objaśniająca (niezależna)
  • a — wyraz wolny
  • b — współczynnik regresji (nachylenie)
Zależność może być liniowa lub krzywoliniowa — tu zajmujemy się liniową.

Interpretacja parametrów

to najczęściej pytają

b — o ile jednostek zmieni się średnio Y, gdy X wzrośnie o jednostkę.

a — wartość Y, gdy X = 0. Gdy X = 0 jest niemożliwe/bez sensu — a pomijamy w interpretacji.

Uwaga na jednostki! X i Y bywają w różnych jednostkach — zawsze dopisz je do interpretacji b.

Metoda najmniejszych kwadratów

skąd biorą się a i b
b = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)² a = ȳ − b·x̄
  • Równoważnie: b = cov(x,y) / S²ₓ
  • MNK minimalizuje sumę kwadratów reszt (odchyleń od prostej)
  • Prosta regresji zawsze przechodzi przez punkt (x̄, ȳ)

Miary dopasowania

jak dobry jest model
determinacja: R² ∈ ⟨0; 1⟩ zbieżność: φ² = 1 − R²
  • R² — jaka część zmienności Y jest wyjaśniona modelem; im bliżej 1, tym lepiej
  • φ² — część niewyjaśniona; im bliżej 0, tym lepiej
  • Dla regresji prostej R² = r² (kwadrat korelacji)
  • s — standardowy błąd szacunku (przeciętny błąd modelu)

Przykład — wydatki a dochód

model z wykładu
ŷ = 8,0871 + 0,0376·x
  • b = 14929,59 / 397483,36 = 0,0376
  • R² = 0,976² = 0,952 → φ² = 0,048
  • Interpretacja b: wzrost dochodu o 1 zł → wydatki na odzież rosną średnio o 0,0376 zł

Regresja wieloraka

wiele zmiennych objaśniających
Y = a₀ + a₁X₁ + a₂X₂ + … + aₖXₖ
  • aᵢ — o ile średnio zmieni się Y przy wzroście Xᵢ o jednostkę, gdy pozostałe X są stałe (ceteris paribus)
  • a₀ — wyraz wolny, zwykle nieinterpretowany
Przykład: sprzedaż pralki Y = 7 605 837 − 4063,38·X₁ + 7606,77·X₂ — cena ↑ o 1 zł → sprzedaż ↓ o 4063,38 zł (przy stałym dochodzie).

Prognozowanie z modelu

po co to wszystko

Podstawiasz przyszłą wartość X do równania i liczysz Y. Warunki, żeby wolno było prognozować:

  • R² > 0,9
  • zmienne objaśniające istotne statystycznie (test t-Studenta)
  • parametry sensowne merytorycznie (znak i wartość)
  • znane przyszłe wartości X
Pułapka jednostek: X podstawiaj w tych samych jednostkach co przy szacowaniu modelu (np. tys. zł), a wynik Y wychodzi w jednostce Y (np. mln zł).

Ocena trafności — błąd ex post

czy prognoza się sprawdziła
błąd = |yₜ − y*ₜ| / yₜ · 100%
0–3% — bardzo dobra
3–5% — dobra
5–10% — dopuszczalna
> 10% — niedopuszczalna
yₜ — wartość rzeczywista, y*ₜ — wartość prognozowana. Błąd ex post liczysz, gdy znasz już rzeczywisty wynik.

Ściąga funkcji Excela

Nachylenie i wyraz wolny to dwie osobne funkcje. Całą tabelę statystyk (parametry, R², błąd standardowy, test t) dostaniesz przez Dane → Analiza danych → Regresja albo funkcją tablicową REGLINP.

Co liczyszFunkcja ExcelaUwaga
Współczynnik b (nachylenie)=NACHYLENIE(Y;X)Kolejność: najpierw Y, potem X
Wyraz wolny a=ODCIĘTA(Y;X)Wartość Y dla X = 0
Współczynnik determinacji R²=R.KWADRAT(Y;X)Lub =WSP.KORELACJI(Y;X)^2
Standardowy błąd szacunku s=REGBŁSTD(Y;X)Przeciętny błąd modelu
Prognoza dla nowego X=REGLINX(nowy_x;Y;X)Podstawia x do prostej regresji
Pełne statystyki (a,b,R²,s,t)=REGLINP(Y;X;PRAWDA;PRAWDA)Funkcja tablicowa (LINEST)
